# MOVVE — The Power Of Precision

Professional logistics company landing page built with a full DESIGN.md design system.

## What's Inside

- **`index.html`** — Full responsive landing page with:
  - Animated particle network hero canvas
  - 9 service cards (Air, Ocean, Ground, Warehousing, Customs, Analytics, Cold Chain, E-Commerce, Project Cargo)
  - Interactive global route map (15 hubs, 28 trade lanes)
  - Animated counters, scroll reveals, 3D card tilt effects
  - Glass-morphism tracking widget
  - Zero dependencies (just Inter from Google Fonts)

- **`DESIGN.md`** — Google DESIGN.md spec (Apache-2.0) with:
  - MOVVE Crimson (#991B1E) accent palette
  - WCAG AAA contrast (8.4:1 on white)
  - Inter typography scale
  - Lint-clean (0 errors via `@google/design.md`)

## Open

Just open `index.html` in any browser — it's fully self-contained.

## Design Tokens

| Token | Value | Role |
|-------|-------|------|
| Primary | `#0A1628` | Navy — authority, text |
| Tertiary | `#991B1E` | MOVVE Crimson — CTAs, accents |
